# AGE OF WAR – Modern JavaScript Remake

A full modern remake of the classic **Age of War** built entirely in **HTML5 Canvas**, **CSS**, and **Vanilla JavaScript**, featuring enhanced visuals, improved gameplay, new difficulty modes, smarter AI, and fully redesigned UI.

This project is open-source and playable directly in the browser.

## About the Project

This is a fan-made recreation of the legendary Flash game **Age of War**, rebuilt from the ground up using modern web technologies.  
It retains the core progression-based gameplay while adding:

- ✔️ New Ages & Units  
- ✔️ Smarter AI with behavior profiles  
- ✔️ Multiple difficulty modes  
- ✔️ Modern UI/UX design  
- ✔️ Particle effects  
- ✔️ Turret upgrades  
- ✔️ Special abilities for each age  
- ✔️ Canvas animations  
- ✔️ Fully responsive interface  

Everything is programmed without external frameworks—just pure JavaScript.

## Features

### Modern AI System
Enemy behavior changes based on difficulty:
- Action timing randomization
- Evolution decision-making
- Threat detection
- Emergency defense logic
- Smart turret building
- AI special-ability usage

### ⚔️ Five Fully Playable Ages
Each age includes unique units, turrets, and a special ability:
1. Stone Age  
2. Castle Age  
3. Renaissance  
4. Modern Age  
5. Future Age  

### Special Abilities
Each age includes a powerful special attack.

### Player Turrets
Up to 3 defensive turrets, each with build cost, upgrade cost, and level scaling.

### Economy System
- Gold per second  
- Gold for kills  
- Difficulty-based multipliers  

## How to Play

1. Open the game in your browser  
2. Choose a difficulty  
3. Train units  
4. Build turrets  
5. Evolve your civilization  
6. Destroy the enemy base  

## Technologies Used

- HTML5 Canvas  
- Vanilla JavaScript  
- CSS3  
- Google Fonts
